# Kasidie City Whisper

A privacy-first iOS app to discover unusual and useful places nearby using OpenStreetMap data.

## Features

- Discover nearby places (museums, art, viewpoints, water fountains, toilets, bike parking)
- Generate 12-minute walking routes
- Offline-first with local caching
- No accounts, no tracking
- Adaptive UI for iPhone and iPad

## Setup

1. Install dependencies:
```bash
flutter pub get
```

2. Generate code:
```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

3. Run the app:
```bash
flutter run
```
